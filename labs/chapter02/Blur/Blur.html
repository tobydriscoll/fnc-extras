Blurb

In a computer, an image is ultimately an array of numbers representing pixel intensity values. We can regard these values as a matrix and model some common image operations using linear algebra.

1. Image import

A color image is imported (in typical cases) as an matrix of red, green, and blue intensity values between 0 and 255. Rather than dealing with full color, we will convert it to an matrix of grayscale intensity values between 0 (black) and 1 (white).
